## 2026-03-03 변경사항

### 1. 데이터베이스 마이그레이션 (Oracle → PostgreSQL)

#### 1.1 연결 문자열 변경
**파일**: `CMS.API/Web.config`

```xml
<!-- 변경 전 (Oracle) -->
<add name="CMS_DB" connectionString="Data Source=(DESCRIPTION=(ADDRESS=(PROTOCOL=TCP)(HOST=222.239.224.42)(PORT=1522))(CONNECT_DATA=(SERVICE_NAME=ORANEW11)));User Id=DID;Password=did1234;" />

<!-- 변경 후 (PostgreSQL) -->
<add name="CMS_DB" connectionString="Host=210.219.229.46;Port=6543;Database=postgres;Username=dbuser;Password=dbuser1234!@#$;SearchPath=publicdata;" />
```

#### 1.2 NuGet 패키지 변경
**파일**: `CMS.API/packages.config`

| 작업 | 패키지 | 버전 |
|------|--------|------|
| 추가 | Npgsql | 3.2.7 |
| 추가 | System.Threading.Tasks.Extensions | 4.3.0 |

> **참고**: .NET Framework 4.5.2 호환성을 위해 Npgsql 3.2.7 버전 사용 (4.x 버전은 .NET 4.6.1+ 필요)

#### 1.3 DB 헬퍼 클래스 생성
**파일**: `CMS.API/App_Code/PostgresHelper.cs` (신규 생성)

- `OracleHelper.cs`를 기반으로 PostgreSQL용 헬퍼 클래스 생성
- 주요 변경 내용:

| Oracle | PostgreSQL |
|--------|------------|
| `using Oracle.ManagedDataAccess.Client;` | `using Npgsql;` |
| `OracleConnection` | `NpgsqlConnection` |
| `OracleCommand` | `NpgsqlCommand` |
| `OracleDataAdapter` | `NpgsqlDataAdapter` |
| `OracleParameter` | `NpgsqlParameter` |
| `OracleDataReader` | `NpgsqlDataReader` |
| `OracleTransaction` | `NpgsqlTransaction` |

- 추가된 메서드 오버로드:
  ```csharp
  public static int ExecuteNonQuery(NpgsqlConnection connection, CommandType commandType, string commandText)
  public static int ExecuteNonQuery(NpgsqlConnection connection, CommandType commandType, string commandText, params NpgsqlParameter[] commandParameters)
  ```

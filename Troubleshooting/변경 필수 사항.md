## 1. 필수 변경 항목

### 1.1 데이터베이스 변경 (Oracle → PostgreSQL)

#### 1.1.1 연결 문자열 변경

**파일 위치**: `CMS.API/Web.config`

```xml
<!-- 현재 설정 (Oracle) -->
<add name="CMS_DB" connectionString="Data Source=(DESCRIPTION=(ADDRESS=(PROTOCOL=TCP)(HOST=222.239.224.42)(PORT=1522))(CONNECT_DATA=(SERVICE_NAME=ORANEW11)));User Id=DID;Password=did1234;" />

<!-- 변경 후 (PostgreSQL 16) -->
<add name="CMS_DB" connectionString="Host=210.219.229.46;Port=6543;Database=postgres;Username=dbuser;Password=dbuser1234!@#$;SearchPath=publicdata;" />
```

#### 1.1.2 NuGet 패키지 변경

```
# 제거
Oracle.ManagedDataAccess

# 설치
Npgsql (PostgreSQL .NET 드라이버)
```

#### 1.1.3 DB 헬퍼 클래스 수정

**파일 위치**: `CMS.API/App_Code/OracleHelper.cs`

- 파일명 변경: `OracleHelper.cs` → `PostgresHelper.cs`
- 클래스명 변경: `OracleHelper` → `PostgresHelper`
- 네임스페이스 변경:
  ```csharp
  // 제거
  using Oracle.ManagedDataAccess.Client;

  // 추가
  using Npgsql;
  ```
- 객체 타입 변경:
  | Oracle | PostgreSQL |
  |--------|------------|
  | `OracleConnection` | `NpgsqlConnection` |
  | `OracleCommand` | `NpgsqlCommand` |
  | `OracleDataAdapter` | `NpgsqlDataAdapter` |
  | `OracleParameter` | `NpgsqlParameter` |
  | `OracleDataReader` | `NpgsqlDataReader` |

#### 1.1.4 SQL 구문 수정 (Biz 폴더 전체)

| 항목 | Oracle | PostgreSQL |
|------|--------|------------|
| 문자열 연결 | `\|\|` | `\|\|` 또는 `CONCAT()` |
| 시퀀스 | `SEQ.NEXTVAL` | `nextval('seq')` |
| NVL 함수 | `NVL(col, val)` | `COALESCE(col, val)` |
| SYSDATE | `SYSDATE` | `CURRENT_TIMESTAMP` |
| ROWNUM | `ROWNUM` | `LIMIT` / `ROW_NUMBER()` |
| 날짜 형식 | `TO_DATE('...', 'YYYY-MM-DD')` | `TO_DATE('...', 'YYYY-MM-DD')` 또는 `'...'::date` |
| 더미 테이블 | `FROM DUAL` | 생략 가능 |

#### 1.1.5 수정 필요 파일 목록


### 1.3 암호화 키 변경

**파일 위치**: `CMS.API/App_Code/CommonProperties.cs` (Line 39)

```csharp
// 현재: 이전 회사명 포함
public static string key = "RETAILTECHSAMSUNGWELSTORY";

// 변경 후 → 새 회사 고유 키로 교체
```

> [!WARNING]

> 새 DB로 시작하거나, 기존 사용자 비밀번호 초기화가 필요합니다.

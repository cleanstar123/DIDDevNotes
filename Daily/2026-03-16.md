## 1. 운영 VM 환경 구축

> [!info] 배경
> - 상부 지시로 개발 서버와 별도의 운영용 VM 필요
> - 동호 프로님께 VM 생성 요청 후 운영 VM 생성 완료

> [!warning] 문제
> - .NET Framework 4.5.1만 설치되어 있어 4.5.2 필요 오류 발생
> - PostgreSQL pg_hba.conf 외부 접속 설정 필요
> - 스키마명 차이 (개발: publicdata, 운영: did)로 DB 마이그레이션 시 추가 작업 필요

> [!success] 해결
> - .NET Framework 4.5.2 Runtime 설치 후 서버 재시작
> - pg_hba.conf에 `host all all 0.0.0.0/0 md5` 추가
> - publicdata 스키마 테이블/함수를 did 스키마로 복사

> [!tip] 배운 점
> - Windows Server에서 IE 보안 강화 구성으로 웹 다운로드 차단됨 → 로컬에서 다운로드 후 복사
> - pg_dump/pg_restore 시 스키마명 변경 필요하면 별도 작업 필요
> - .NET Framework 버전 확인: `reg query "HKLM\SOFTWARE\Microsoft\NET Framework Setup\NDP\v4\Full" /v Release`
